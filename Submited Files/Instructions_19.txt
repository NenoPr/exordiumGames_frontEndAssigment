I have used a "Live Server" plugin through VS code for development.

To best run the html files, a live server is required to be opened in the root folder, 
then clicking on the folders "season" or "gameIndex" will launch their respective index.html files. 

Opening the html.file by itself will not work on the seasons index.html file due to the CORS policy blocking the local path GET requests.
